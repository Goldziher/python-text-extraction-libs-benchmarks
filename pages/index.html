🔧 Framework Details

Kreuzberg

Version: {versions['kreuzberg']} | Size: 71MB base

Fast Python text extraction with multiple OCR backends. Supports both sync and async APIs.

Strengths: Speed, small footprint, async support

Limitations: No email/data format support (by design)

Docling

Version: {versions['docling']} | Size: 1GB+

IBM Research's advanced document understanding with ML models.

Strengths: Advanced ML understanding, high quality

Limitations: Large size, slower on complex PDFs

MarkItDown

Version: {versions['markitdown']} | Size: 251MB

Microsoft's lightweight Markdown converter optimized for LLM processing.

Strengths: LLM-optimized output, ONNX performance

Limitations: Limited format support

Unstructured

Version: {versions['unstructured']} | Size: 146MB

Enterprise solution supporting 64+ file types.

Strengths: Widest format support, enterprise features

Limitations: Moderate speed

Extractous

Version: {versions['extractous']} | Size: ~100MB

Fast Rust-based extraction with Python bindings.

Strengths: Native performance, low memory usage

Limitations: Newer library, some format gaps
